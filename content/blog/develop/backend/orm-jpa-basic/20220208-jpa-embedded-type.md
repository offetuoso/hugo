# 임베디드 타입
--------------------------------

## 임베디드(복합 값) 타입
--------------------------------
> - 새로운 값 타입을 정의할 수 있음
> - JPA는 임베디드 타입(embedded type)이라 함
> - 주로 기본 값 타임을 모아 만들어서 복합 값 타입이라고도 함
> - int, String과 같은 값 타입
